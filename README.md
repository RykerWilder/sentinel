### Dependencies

1. **Create venv**
```zsh
python3 -m venv venv
```

2. **Activate venv**

MacOS/Linux
```zsh
source venv/bin/activate
```
Windows
```zsh
venv\Scripts\activate
```

3. **Install project dependencies**
```zsh
pip install -r requirements.txt
```
