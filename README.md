### Dependencies

# 1. Crea e attiva il venv
python -m venv venv
source venv/bin/activate  # o venv\Scripts\activate su Windows

# 2. Installa le dipendenze del progetto (se presenti)
pip install -r requirements.txt

Now to start Sentinel:
```bash
sentinel start
```
